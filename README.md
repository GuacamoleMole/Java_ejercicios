# EjerciciosJava
Este repositorio contiene dos ejercicios de Java:

## Ej2

- **Descripción:**
	Simula un sistema de pedidos de restaurante. Modela diferentes tipos de platos (`Pizza`, `Filete`) y métodos de preparación (`Frito`, `Hervido`, `Horneado`). El programa principal crea varios platos, los añade a una orden y la prepara.
- **Tecnologías:**
	- Java 21
	- Maven
	- Usa Lombok y SLF4J para logging
- **Punto de entrada:**
	- `Ej2/untitled/src/main/java/org/example/Main.java`

## Ej3

- **Descripción:**
	Implementa un sistema bancario concurrente. Crea varias cuentas bancarias y realiza operaciones aleatorias (depósito, retiro, transferencia) de forma concurrente usando virtual threads y `CompletableFuture`.
- **Tecnologías:**
	- Java 21
	- Maven
- **Punto de entrada:**
	- `Ej3/Ej3/src/main/java/org/example/Main.java`
